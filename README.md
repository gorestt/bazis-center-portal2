# Bazis Center Portal

Веб-приложение **«Система мониторинга IT-сервисов и бизнес-процессов ООО «Базис-Центр»**  
(рабочее название — **«Панель мониторинга»**).

Проект реализован на **Django** и предназначен для мониторинга очереди заявок, KPI, инцидентов, смен персонала и отчётности, а также для предоставления публичного интерфейса для клиентов и менеджмента.

---

## Основные возможности

- Публичная витрина показателей и информации:
  - главная страница с кратким описанием сервиса и KPI;
  - публичная очередь заявок;
  - публичные KPI (Chart.js);
  - публичный список инцидентов;
  - публичный график смен;
  - публичный список сформированных отчётов;
  - раздел документации с возможностью скачивания файлов;
  - FAQ (не менее 10 вопросов–ответов);
  - страницы «О компании», «Контакты», «Сервисы», «Новости».

- Закрытая часть (панель мониторинга):
  - панель сводных показателей;
  - очередь заявок (CRUD, фильтры, пагинация);
  - панель KPI с графиками (Chart.js) и JSON API;
  - список инцидентов;
  - график смен персонала;
  - панель отчётов с генерацией файлов (.docx / .txt);
  - управление документацией, загрузка файлов (.docx / .pdf / .xlsx);
  - личный кабинет клиента с его заявками.

- Работа с файлами и отчётами:
  - загрузка регламентов и инструкций в разделе «Документация»;
  - автоматическая генерация отчётов за период;
  - хранение файлов в `media/`.

- Роли и разграничение доступа:
  - **admin** — полный доступ, управление пользователями и ролями;
  - **manager** — работа с очередью, KPI, инцидентами, сменами, отчётами и документацией;
  - **client** — личный кабинет, просмотр собственных заявок и публичной информации.

---

## Технологии

- **Backend**: Django (Python)
- **База данных**: SQLite (по умолчанию), возможность перехода на PostgreSQL
- **Фронтенд**: Bootstrap 5, Chart.js
- **Web-сервер (продакшн)**: gunicorn
- **Генерация документов**: `python-docx`
- **Хостинг **: Render.com
